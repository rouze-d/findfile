FINDFILE : SIMPLE TOOL FOR FILE FINDER

Findfile is simple tool for try to find file include: log, modifed, backup, ssistant's workstation, etc... 

need:<br>
```
sudo gem install lolcat
sudo apt install curl
```

usage:<br>
```
bash findfile.sh
```
